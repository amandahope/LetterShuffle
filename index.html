A JavaScript-Based Word Game

Directions:

  • Click "New Game" to start a game.
  • Type words 2-6 letters long that you can make from the 6 letters given.
  • Press Enter or click Submit after each word. If the word is valid, it will be added to the list. If not, you'll get an error.
  • To rearrange the letters during the game, click Shuffle.
  • The game ends after 3 minutes.
  • High scores will be saved in the browser.
The word lists used to power this game are provided by WordGameDictionary.com
Layout is based on the Skeleton CSS framework.
